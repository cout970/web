Kotlin libraries

Github repo

A set of useful libraries for 3D applications and OpenGL. Contains the following libraries:

  • CollisionLib: Collision detection for 2D and 3D shapes
  • GlUtilities: Boilerplate code to deal with OpenGL
  • MatrixLib: Implements mutable and immutable matrices of 2x2, 3x3 and 4x4
  • VectorLib: Implements mutable and immutable vectors of 2, 3 and 4 elements and quaternions
  • OblLib: Simple Obj format loader
  • RaytraceLib: Raytrace of simple 3D geometries

Right now the libraries are being ported to Kotlin Multiplatform to run in Native, JVM and JS with minimal changes. The first library ported is VectorLib

3D Modeler

Github repo

An application to create and edit 3D models, supports obj, glTF and several minecraft-especific formats. Its primary goal was to replace an old tool called Techne that allowed to create minecraft models using cubes and planes. This tools has the same functionalities but adds a bunch more like 3D animation, use of any kind of shape, custom UV mapping and more.

Reactive

Github repo

A copy of React-JS made on top of Legui (a.k.a. SpinyOwl), allows to make user interfaces in Kotlin in the same way react does. You make a render function that transforms the state of you application into an tree of components called virtual DOM, the library compares the current tree of components and the new tree of components and only updates the components that are different keeping the rest intact.
